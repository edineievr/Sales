preço vs valor

preço é quanto custa, conceito quantitativo
valor é mais perceptivo, emocional, conceito abstrato

nomenclaturas de testes exemplos:
when/should
- when-condition-should-expecetedresult

method_shouldoutcome
- method-should-expectedresult

condition-shouldoutcome
- emptyorder_should_havezerototal

Notes:
Exceção responde “qual regra foi violada”.
Validator valida dados.
Domínio valida regras.
Construtor garante existência válida.
Métodos garantem transições válidas.
State validation happens when state is created or changed
Calculations assume state is already valid
